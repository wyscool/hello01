Python 简介

Python 是一门解释型、动态类型的编程语言，由 Guido van Rossum 于 1991 年发布。

核心特性：
1. 简洁易读的语法 —— 用缩进定义代码块，不用花括号
2. 动态类型 —— 变量不需要声明类型，运行时自动推断
3. 丰富的标准库 —— "自带电池"，涵盖网络、文件、数据处理等领域
4. 多范式 —— 支持面向对象、函数式、过程式编程

Python 广泛应用于：Web 开发（Django、Flask）、数据科学（pandas、numpy）、
机器学习（PyTorch、scikit-learn）、自动化运维、爬虫等领域。

Python 3.12 引入了更详细的错误信息、改进的 f-string 语法、
类型提示的简化写法（PEP 695）等重要特性。
