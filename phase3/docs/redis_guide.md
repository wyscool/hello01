# Redis 入门指南

## 什么是 Redis

Redis 是一个开源的内存数据结构存储系统，常用作数据库、缓存和消息代理。

## 核心数据结构

1. **String** —— 字符串，最基础的类型，可以存数字、JSON
2. **Hash** —— 哈希表，适合存对象
3. **List** —— 链表，适合消息队列
4. **Set** —— 无序集合，支持交并差集运算
5. **Sorted Set** —— 有序集合，按 score 排序

## 常见使用场景

- 缓存热点数据，减轻数据库压力
- 分布式锁（SETNX + 过期时间）
- 计数器、排行榜（Sorted Set）
- 消息队列（List + Pub/Sub）
- 限流器（滑动窗口）

## 持久化

Redis 支持两种持久化方式：
- RDB：定时快照
- AOF：追加写日志

生产环境建议同时开启两种，用 RDB 做全量恢复，AOF 保证增量不丢。
